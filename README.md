# What the tool does
This tool tracks the user's profile information like weight, height, sex, age, and goal weight and creates a schedule for muscles worked by day. The tool also shows weight progress in a graph once inputed and includes an exercises library of sample exercises with information about equipment, reps, and experience level.

# How to use it
The user is to put in their information in the profile page and they have the option to save their profile or not. The user can then go over to the schedule page and week schedule will be provided based on the amount of days the user wants to workout. The user can go over to the plan page and see the amount of calories they should burn each day with BMR and TDEE calculations. The user can also input their weight along theire weightloss journey and see trends in the graph. Lastly, there is an exercise library where users can look up exercises as well as put their own.
