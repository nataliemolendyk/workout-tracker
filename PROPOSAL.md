## What I'm building:
I am building JavaScript tool that runs on GitHub Pages and it allows the user to input their height, age, weight, sex, goal weight, goals for working out, and it gives the user their plan for weight loss based on calculations as well as recommended exercises for their goals.

## Who it's for / why: name a real person or yourself; what problem does it solve?
This is for me but also anyone else looking to potentially lose/gain any amount of weight and they don't know where to start. The problem that this solves is it takes the stress out of creating a plan without prior knowledge about health or fitness.

## The state it tracks: what data does the tool need to remember?
The tool would need to remember (but is optional) the user's profile information, preferences, and cache (past results).

## Core features: (3-5 things it will do)
- Provide exercise recommendations
- Calculate BMR/TDEE results
- Calculate estimated timeline to goal weight
- Generate a workout plan

## What I don't know yet: which JavaScript concepts will you need to figure out?
- Conditionals that direct the user to the correct next actions
- Objects and arrays for the user profile, workout plan, and exercise library
- Getting the website to feel like an app (DOM)
- Workout templates or exercise database using JSON
- Generating schedules
